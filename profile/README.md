# AInterviewer

Welcome to the AInterviewer project's GitHub organization!

Here you will find a collection of different software tools developed to be used for conducting large scale AI-led qualitative interviews, and repositories for various related empirical and analytical projects.

---

The software is separated into three different repositories:  
1. Core [library](https://github.com/ainterviewer/lib): The core logic of the AInterviewer is specific here with general protocols to interact with different interfaces  
2. Platform [backend](https://github.com/ainterviewer/lib): FastAPI backend with database and api configuration  
3. Platform [frontend](https://github.com/ainterviewer/lib): SvelteKit frontend with UI to setup, manage, distribute and run AInterviewer

> [!warning]
> All three software projects are still in early stages, so expect frequent and breaking changes.
